# Cardiometabolic Multimorbidity Batch Risk Assessment Tool

A machine learning-based risk assessment tool for cardiometabolic multimorbidity.

Focuses on predicting CVD in individuals aged 65 and older with metabolic diseases.

Specifically designed for community public health workers.

## Tool Usage Scenarios

Suitable for routine screening tasks at community health service centers, including： 

⦁ batch risk assessment during annual community health checkups； 

⦁ periodic screening and follow-up for high-risk populations.

# Technology Stack

* Streamlit
* Scikit-learn
* LightGBM
* Pandas \& NumPy
